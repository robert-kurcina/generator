Military Generation

The Military.js module provides the attribute to generate a character of quality q or with dice modifier m. The syntax is as follows: Military.getCharacter(); //generate a completely random character
Military.getCharacter(n); //generate a character using dice roll n as the table index;
Military.getCharacter(null, m); //generate a random character but add m to the dice roll. null must be the first parameter
Pass in a '*' for n to create a Star character. The module otherwise will generated a character quality at random. The result object has a toString() method to disclose its contents, but these can also be accessed via members as: Military.getCharacter().toString(); //disclose the result object
Military.getCharacter().quality; //what is the quality of the character
Military.getCharacter().rep; //what is the rep of the character
Military.getCharacter().weapon; //what is the weapon possessed by the character
